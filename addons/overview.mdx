---
title: "Addons"
description: Extend TREK with optional modules — enabled and managed from the Admin Panel.
---

Addons are optional modules that extend TREK's core functionality. Each addon can be independently enabled or disabled by an administrator, so teams can tailor the feature set to their needs without affecting the rest of the application.

## Enabling and disabling addons

<Steps>
  <Step title="Open the Admin Panel">
    Log in as an administrator and navigate to **Admin Panel** from the user menu.
  </Step>
  <Step title="Go to Addons">
    Select the **Addons** section from the sidebar.
  </Step>
  <Step title="Toggle the addon">
    Enable or disable any addon using its toggle. Changes take effect immediately — no restart required.
  </Step>
</Steps>

<Note>
  Only administrators can enable or disable addons. Regular users see menu items only for addons that are currently enabled.
</Note>

## Available addons

<CardGroup cols={2}>
  <Card title="Atlas" icon="globe" href="/addons/atlas">
